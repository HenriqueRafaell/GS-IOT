# 🌙 Guardião de Habitats Extremos

Sistema de detecção de anomalias atmosféricas usando TinyML em ESP32.

## 👥 Integrantes
- HENRIQUE RAFAEL
- RM:553945

## 🎯 Problema
Em habitats lunares, despressurização e radiação são riscos críticos.
Este sistema detecta anomalias em tempo real com latência < 100ms.

- TinyML (Edge Impulse)
- Arduino C++
- Webserver integrado

## 📊 Resultados
- Acurácia: 96.7%
- Latência: < 10ms
- 3 Estados: Seguro, Alerta, Crítico
